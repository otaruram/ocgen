# OCGen AI - The Ultimate AI Generator for VS Code

Transform your workflow with **OCGen AI**, the premium Visual Studio Code extension designed to generate pixel-perfect SVG icons and robust JSON mock data directly within your editor. Built for modern developers, OCGen AI leverages advanced AI models to accelerate your front-end and back-end development effortlessly.

## 🚀 Core Functionalities

- **AI-Powered SVG Generation**: Instantly create customized, clean, and minimalist SVG icons by simply describing what you need. 
- **Dynamic JSON Mock Data**: Generate realistic, structurally sound JSON data for APIs, user profiles, e-commerce products, and more in seconds.
- **One-Click Code Insertion**: Preview your generated SVGs and JSON arrays/objects, then seamlessly insert them right at your active cursor position.
- **Context-Aware Ideation**: OCGen AI understands your prompts to deliver tailored assets that match your project's aesthetic and data requirements.
- **Premium Native UI**: Flawlessly integrates with VS Code's Light and Dark themes, offering a distraction-free and highly responsive sidebar experience.

## 🧠 Model & BYOK (Bring Your Own Key)

OCGen AI operates on a highly secure **BYOK (Bring Your Own Key)** model, ensuring enterprise-grade privacy and full control over your usage. 

We utilize the **Sumopod API** to power our generation engines. By integrating cutting-edge AI via Sumopod, we guarantee rapid inference, high-quality vector outputs, and intelligent data structuring. 

To get started:
1. Obtain your API key from [oclite.site/get-key](https://oclite.site/get-key).
2. Input and save your key securely in the OCGen AI sidebar. It is safely stored in your local VS Code SecretStorage.
3. Start generating without limits!
