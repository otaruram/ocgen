# OCGen AI: SVG, PNG & JSON Mock Data Generator for VS Code

Transform your development workflow with **OCGen AI**, the ultra-lightweight (only 58kb!) and premium **VS Code AI extension**. Designed to generate pixel-perfect SVG icons, minimalist PNG assets, and robust JSON mock data directly within your editor. 

Built to serve as your ultimate **AI code assistant** and **UI mockup tool**, OCGen AI leverages advanced AI models to accelerate your front-end and back-end development without bloating your IDE.

## 🚀 Core Functionalities

- 🎨 **AI SVG Generator**: Instantly create customized, clean, and scalable vector SVG icons by simply describing what you need. Perfect for modern web design.
- 🖼️ **AI PNG Generator**: Need raster graphics? Easily generate **minimalist PNG icons** natively without leaving VS Code.
- 📊 **Dynamic JSON Mock Data**: Generate realistic, structurally sound **dummy data** for APIs, user profiles, e-commerce products, and databases in seconds.
- ⚡ **One-Click Code Insertion**: Preview your generated UI assets visually in the sidebar, then seamlessly insert the raw code right at your active cursor position.
- 🧠 **Context-Aware AI Assistant**: OCGen AI understands your prompts to deliver tailored developer tools that match your project's aesthetic and data requirements.
- 🌗 **Premium Native UI**: Flawlessly integrates with VS Code's Light and Dark themes, offering a distraction-free and highly responsive sidebar experience.

## 🔒 Privacy-First: BYOK (Bring Your Own Key)

OCGen AI operates on a highly secure **BYOK (Bring Your Own Key)** model, ensuring enterprise-grade privacy, zero data retention, and full control over your usage costs.

We utilize the lightning-fast **Sumopod API** to power our generation engines. By integrating cutting-edge **AI for VS Code**, we guarantee rapid inference, high-quality vector outputs, and intelligent data structuring that rivals premium AI code generators at a fraction of the cost.

### How to Get Started:
1. Obtain your API key from [Sumopod](https://sumopod.com).
2. Input and save your key securely in the OCGen AI sidebar. It is encrypted and safely stored in your local VS Code SecretStorage.
3. Start generating UI assets and mock APIs without limits!
